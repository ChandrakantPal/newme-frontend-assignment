## Newme Frontend Task

Add OMDB API in the `.env` or `.env.local` file

````ts
NEXT_PUBLIC_API_KEY=YOUR_API_KEY
````

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
````

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Stack

- Next.js
- TailwindCSS
- OMDB API
